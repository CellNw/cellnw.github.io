An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale

Szl

CVPR 2020

摘要

当 Transformer 结构成为自然语言处理的标准时，它在机器视觉方面的应用是受限的。在机器视觉领域注意力要么与卷积神经网络组合使用，要么替换卷积网络的某些组件同时保存其整体网络结构。我们表明，这种对 CNN 的依赖是不必要的，直接应用图像补丁序列的纯转换器可以在图像分类中表现得很好。

1 简介

介绍了一下注意力机制在自然语言处理上的应用，指出 Transformer 拥有卓越的性能。

本文尝试直接将标准Transformer 应用于图像，尽可能得少修改。为此，我们将图像分割为小块，并将这些小块的线性嵌入序列作为 Transformer 的输入。图像块已与 NLP 应用程序中的标记相同的方式处理。并发现 ViT 在中型规模的数据集上不如 ResNet ，但在大型数据集比如 ImageNet-21k 上超过最先进的水平。

2 相关工作

指出以前的 Transformers 不能很好得在已有的加速硬件上工作，或者只能在低分辨率图片上使用。


图1:模型的整体框架。我们首先将图片切分成若干块（patches），再线性得嵌入它们并添加位置编码。然后再将它们当作向量序列放入标准的 Transformer 编码器中。为了执行分类任务，我们使用标准方法：增加可学习的”分类标识（classification token）”到序列中。

3 方法

在模型设计中，尽可能遵循原始 Transformer。

3.1 Vision Transformer（ViT）

图1是模型的整体框架。标准 Transformer 的输入是一个一维嵌入 token 的序列。为了处理二维的图片，我们把图像 $x\in\mathbb{R}^{H\times W\times C}$ 重塑成一个被展平的二维补丁序列（patches）$x_{p}\in\mathbb{R}^{N\times(P^{2}\cdot C)}$ ，其中 $(H,W)$ 是原始图像的分辨率，$C$ 是通道数，$(P,P)$ 是图像补丁的解析度。并且 $N=HW/P^{2}$ 是结果的补丁（patches）数量，这也作为 Transformer 输入的有效序列长度。The Transformer uses constant latent vector size D through all of its layers, so we flatten the patches and map to D dimensions with a trainable linear projection (Eq. 1).我们将此投影的输出称为补丁嵌入。

与 BERT 的 [class] 令牌相似，我们在嵌入补丁的序列中预置一个可学习的嵌入（$Z_{0}^{0}=X_{\text{class}}$）,其在 Transformer 编码器的输出状态（$Z_{L}^{0}$）用作图像表示 $y$ 。在预训练和微调期间，分类头都附加到$Z_{L}^{0}$ 。分类头在预训练时由具有一个隐藏层的 MLP 实现，在微调时由单个线性层实现。

位置嵌入被添加到 patch embedding 以保留位置信息。使用标准的可学习一维位置嵌入，因为没有观察到更高级的二维感知位置嵌入的显著性能提升。生成的嵌入向量序列用作编码器的输入。

Transformer 编码器由交替的多头自注意力（MSA）和 MLP 块组成。Layernorm（LN）在每隔块之前应用，每个块之后应用残差连接。

归纳偏置 我们注意到 ViT 比 CNN具有更少的图像特定的归纳偏置，在 CNN 中 局部性、二维领域结构和平移不变包含在模型的每一层中。在 ViT 中，只有 MLP 层具有局部性和平移不变性。然而自注意力机制是全局的。二维邻域结构仅在模型开始时，把图像分割成 patches 和在微调时调整不同分辨率的图像位置嵌入时使用。

混合架构 作为原始图像补丁的替代方案，输入序列可以由 CNN 的特征图形成。在这个混合模型中，补丁嵌入投影 E （公式1）应用于从 CNN 特征图提取的补丁。

3.2 微调和更高的分辨率

典型的情况是我们在大型的数据集上预训练 ViT 然后对它进行微调去适应（更小的）下游任务。我们去掉了预训练好的预测头，并加入了一个零初始化的 $D\times K$ 前馈层。其中 $K$ 是下游任务的分类数量。使用比预训练时更高的分辨率进行微调是有益的。当输入的图像具有更高的分辨率时，我们保持 patch 的大小不变，从而产生了更大的有效序列长度。ViT 可以处理任意的序列长度（直到内存限制），但是，预训练好的位置嵌入可能不再有意义了。因此，我们根据它们在原始图像中的位置对预训练的位置嵌入进行二维插值。请注意，这种分辨率调整和补丁提取是手动将图像二维结构的归纳偏置注入 ViT 的唯一点。
