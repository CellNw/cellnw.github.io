Fixing Data Augmentation to Improve Adversarial Robustness

Abstract 本文主要关注如何使用启发式驱动增强（Pad & Crop，Cutout，MixUp，CutMix）和数据式驱动增强（VDVAE，BigGAN，DDPM）提升模型鲁棒性。 本文通过组合使用 model weight averaging 和 data augmentation 大幅提高了鲁棒准确性。 本文探索了如何利用最先进的生成模型来人为地增加数据集大小。 最终在 $\epsilon=8/255$ 的 $l_\infty$ 和 $\epsilon=128/255$ 的 $l_0$ 范数的约束的对抗样本攻击下，比 SOTA 方法分别提升了 $7.06%$ 和 $5.88%$ 。 介绍了一下对抗样本，并指出对抗训练是目前最有效的防御方法。 本文贡献： 展示了当与 model weight averaging 结合时，启发式驱动的增强技术，如 Cutout、CutMix 和 MixUp 如何提高鲁棒性。 与前人尝试了数据增强但是都没有成功相反，本文能够通过上述三种技术的任意一种来获得新的 SOTA 鲁棒性精度。并发现 CutMix 是最有效的方法，它在 CIFAR10…
