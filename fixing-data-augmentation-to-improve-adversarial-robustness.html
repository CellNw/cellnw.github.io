Fixing Data Augmentation to Improve Adversarial Robustness

Szl

Abstract

本文主要关注如何使用启发式驱动增强（Pad & Crop，Cutout，MixUp，CutMix）和数据式驱动增强（VDVAE，BigGAN，DDPM）提升模型鲁棒性。

本文通过组合使用 model weight averaging 和 data augmentation 大幅提高了鲁棒准确性。

本文探索了如何利用最先进的生成模型来人为地增加数据集大小。

最终在 $\epsilon=8/255$ 的 $l_\infty$ 和 $\epsilon=128/255$ 的 $l_0$ 范数的约束的对抗样本攻击下，比 SOTA 方法分别提升了 $7.06%$ 和 $5.88%$ 。

1. Introduction

介绍了一下对抗样本，并指出对抗训练是目前最有效的防御方法。

本文贡献：

  • 展示了当与 model weight averaging 结合时，启发式驱动的增强技术，如 Cutout、CutMix 和 MixUp 如何提高鲁棒性。

  • 与前人尝试了数据增强但是都没有成功相反，本文能够通过上述三种技术的任意一种来获得新的 SOTA 鲁棒性精度。并发现 CutMix 是最有效的方法，它在 CIFAR10 上达到了 $60.07%$ 的鲁棒准确率，扰动为$l_\infty ,\epsilon=8/255$ （在原先 SOTA 上提升了 $2.93%$）。

  • 解释了数据驱动的增强如何帮助提高多样性并补充启发式驱动的增强。利用额外产生的输入（仅使用在原始数据上训练的生成模型产生的输入），并研究了三个最近的生成模型：去噪扩散概率模型（DDPM）、深度变分自动编码器（VDVAE）和 BigGan。

  • 发现由 DDPM 生成的图像信息量更大，使我们能够在 CIFAR-10 上达到 $63.58%$ 的鲁棒准确性，扰动为$l_\infty ,\epsilon=8/255$ （在原先 SOTA 上提升了 $6.44%$）。

  • 最后，本文结合了启发式和数据驱动式的方法。最终在 CIFAR-10 上，在 $\epsilon=8/255$ 的 $l_\infty$ 和 $\epsilon=128/255$ 的 $l_0$ 范数的约束的对抗样本攻击下，比 SOTA 方法分别提升了 $7.06%$ 和 $5.88%$ 。并且我们最好的模型击败了所有使用额外数据的技术。

2. Related Work

对抗 $l_p$ - 约束攻击：介绍了一下 FGSM，R+FGSM 和 BIM。

使用对抗训练进行防御：介绍了一下对抗训练。

启发式数据增强：指出数据增强已被证明可以减少标准（非鲁棒）训练的泛化误差，但它们在训练对抗性鲁棒网络时仍然无效。

数据驱动式数据增强：介绍了两种图像生成方法 AutoAugment 和 RandAugment。

4. Heuristics-driven Augmentations

在没有外部数据的情况下限制鲁棒性过拟合：前人表明，将 Cutout 或 MixUp 等方法与 early stopping 结合并不能提高鲁棒性。虽然这些方法并没有提升最佳精度但是减少了鲁棒性过拟合的程度。见图三。


图3:在使用 MixUp 或随机填充和裁剪（Pad & Crop）的标准组合时，使用和不使用模型权重平均 (WA) 时，在 CIFAR-10 上针对 ∞ = 8/255 的准确度。该模型是 WRN-28-10，该图显示了针对 PGD40 的测试集的稳健精度
**验证假设**：比较了 Pad & Crop 和 MixUp 分别添加和不添加 WA 的情况发现，WA 均可以缓解鲁棒性过拟合。如图三。

探索启发式驱动的增强：在验证了 MixUp 能够维持鲁棒性精度，并在与 WA 结合后能够提升精度后，本文测试了其他图像补丁技术：Cutout 和 CutMix。在 6.2 节作者还评估了 AutoAugment、RandAugment 和 RICAP。

5. Data-driven Augmentations

多样性和互补性：虽然启发式驱动的增强显著地提升了鲁棒性。但是和使用了其他数据的防御模型相比还是有很大的差距。一个合理的解释时：常见的增强技术往往会产生接近它们原始图片的增强图片，这意味着在提高泛化能力方面受到了限制。

生成模型：介绍了一下 BigGan、VDVAE 和 DDPM。实验发现 DDPM 与 80M-TI （真实、非生成图像）的分布更接近。BigGAN 和 VDVAE 生成的图像与其各自生成的图像更加接近，这表明这些样本要么远离训练和测试分布，要么产生过于相似的样本。

数据驱动的增强对鲁棒性的影响：作者测试发现只有 DDPM 生成的样本成功地防止了鲁棒的过拟合，并在学习率下降后导致稳定的性能。

6. Experimental Results

6.1 实验设置

神经网络结构：使用 WRNs，WA 的衰减率分别设为 $\tau=0.999$ 和 $\tau=0.995$ ，如果没有特别说明，实验是在深度为 $28$ 且宽度乘数为 $10$ 的 WRN-28-10 的模型上进行，包含 $36$M 的参数。

对抗训练：本文使用 TRADES ，$10$ PGD。在没有添加其他数据时训练 400 epoch，batch size 为 512。在使用额外数据时 batch size 增加到 1024，原始数据与添加数据的比率为 0.3，并训练 800 个 CIFAR-10 等效的 epoch。

评估：遵循 Gowal 等人涉及的评估协议。（Uncovering the Limits of Adversarial Training against Norm-Bounded Adversarial Examples）

6.2 启发式增强

使用此前的 SOTA （Pad & Crop）作为比较基准，本文比较了 MixUp，Cutout，CutMix 和 RICAP。结果如图四。


图4:使用不同的启发式增强方法在干净样本准确率和鲁棒性准确率。
在图片上可以清楚看到三个集群。第一个集群是由 AutoAugment 和 RandAugment 组成，它们提升了在干净样本上的准确率但是在鲁棒准确率上降低。第二个集群是包括 RICAP、Cutout 和 CutMix 包含三种方式用补丁遮挡局部信息。最后一个集群 MixUp 只相比基准提升了 $0.91\%$ 。一个合理的解释是 MixUp 当 $\alpha$ 很大的时候产生原理原始数据的分布或者当 $\alpha$ 较小时产生与原始样本太接近的分布。
6.3 数据驱动增强

讲了一些实验设置，使用不同生成方法（VDVAE、BigGAN 和 DDPM）生成图像，并将这些生成图像按照不同的比例与原始图像混合，最终实验效果如图5。


图5:作者比较了训练批次中原始图像与生成图像之间的比率如何影响测试鲁棒性能。
最终发现使用 DDPM 生成方法并且比率为 0.3 时鲁棒性提升最大（$+6.29\%$）。而BigGAN 和 VDVAE 仅相较于基准提升了 $1.55\%$ 和 $1.07\%$ 。
6.4 组合所有的增强方法

使用 CutMix 加 DDPM 最终在 $l_{\infty}$ 的设置下提升了 $6.44%$，在 $l_{2}$ 的设置下提升了 $3.81%$ 。

7. Conclusion

我们假设并证明生成的样本提供了更大的增强多样性，有助于密集图像流形并允许对抗训练远远超出了最先进的技术。我们的工作为模型权重平均对鲁棒性的影响提供了新的见解，我们希望进一步了解鲁棒性。
